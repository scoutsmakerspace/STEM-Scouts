# STEM Scouts

A Jekyll + Minimal Mistakes website for collecting STEM activities for Scouts (Beavers/Cubs/Scouts/Explorers), including standardised Risk Assessments and badge requirement links.

## Editing
- Content editors: visit `/admin/` on the deployed site (Decap CMS).
- Activities live in `_activities/`
- Risk assessments live in `_risk_assessments/`
