# Public Map and Statistics Model — Maker Kits

## Default map model

The default public map should be aggregate-only.

Show:

- postcode district
- approximate public area
- number of groups/public map entries supported
- kit count band

Do not show:

- full address
- full postcode
- customer/contact name
- email address
- payment status
- private issue records
- exact order rows
- packing data

## Optional future consent model

A later version may show public supporter names only when an explicit `public_map_consent` field is true.

Until then, group names should stay out of the map export.

## Public statistics

Good public headline statistics:

- kits supplied
- groups supported
- postcode districts reached
- batches completed
- estimated young people reached
- broad volunteer effort hours, if comfortable

Where estimates are used, explain the assumption.

Example:

> Young people reached is estimated from kits supplied. It is not a register of individual children.
