# Future Product Concept — Community Kit Fulfilment Tool

## Vision

Turn the current Scout/charity kit workflow into a free, reusable fulfilment tool for charities, clubs, schools and community groups that make and distribute physical kits.

## Current position

The current Google Sheets / Apps Script tool remains the operational back end. It already manages orders, batches, components, packing logic, assignments, emails, issues, effort, reports, backups and archive.

## Future generic entities

A generic version should separate the specific Scout kit operation from reusable fulfilment concepts:

- kits
- components
- suppliers
- batches
- orders
- packing modes
- work assignments
- issues
- communications
- stock/material movements
- public exports
- reports and archive

## Possible free routes

1. Keep a Google Sheets / Apps Script template for small groups.
2. Build a local/offline desktop helper for heavier packing workflows.
3. Build a static public website front-end fed by sanitised exports.
4. Later, consider a lightweight web app only if hosting and privacy can remain practical and free.

## Product rule

The public website/front-end is not the fulfilment system. It is the public window into a private, controlled operational workflow.
